Compiler Option: -lib
 
Create a static library

Syntax

-lib

Description

The -lib compiler option creates a static library.

The intrinsic macro __FB_OUT_LIB__ is set to non-zero (-1) if the -lib option was specified, and set to zero (0) otherwise.
