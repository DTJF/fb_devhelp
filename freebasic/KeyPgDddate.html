__DATE__
 
Intrinsic define (macro value) set by the compiler

Syntax

__DATE__

Description

Substitutes the compiler date in a literal string ("mm-dd-yyyy" format) where used.

Example

Print "Compile Date: " & __DATE__


Compile Date: 09-29-2011
