__FB_ARM__
 
Intrinsic define set by the compiler

Syntax

__FB_ARM__

Description

Define created at compile time if the compilation target uses the ARM CPU architecture, otherwise undefined.

Example

#ifdef __FB_ARM__
  '...instructions for ARM OSes...
#else
  '...instructions for other OSes
#endif
