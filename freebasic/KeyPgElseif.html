ElseIf
 
Control flow statement for conditional branching

Syntax

If expression Then
[statement(s)]
[ ElseIf expression Then ]
[statement(s)]
[ Else ]
[statement(s)]
End If


Remark: EndIf (without blank) is also supported like in QB for backward compatibility.
