#inclib
 
Preprocessor directive

Syntax

#inclib "libname"

Description

Includes a library in the linking process as if the user specified -l libname on the command line.

Example

'' incomplete code snippet

'' this will include libmystuff.a in the link process
#inclib "mystuff"
