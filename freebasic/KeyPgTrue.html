True
 
Intrinsic constant set by the compiler

Syntax

Const True As Boolean

Description

Gives the True Boolean value where used.

Example

Dim b As Boolean = True
If b Then
    Print "b is True"
Else
    Print "b is False"
End If


b is True

Dialect Differences

  • Not available in the -lang qb dialect unless referenced with the alias __True.

Differences from QB

  • New to FreeBASIC
